# Prompt: Write API Documentation (Login Endpoint)

Write API documentation for a "User Login" endpoint.

## Requirements:
- Include:
  - Endpoint URL
  - HTTP Method
  - Headers
  - Request Body
  - Response Body
  - Error Codes (400, 401)
- Provide sample request and response in JSON
- Keep tone professional and concise

## Constraints:
- Use clear section headings
- Avoid unnecessary explanations
- Keep total length under 250 words

## Output Format:
- Markdown with headings
- JSON code blocks for request/response
