# Prompt: Explain REST APIs (Beginner-Friendly)

Explain REST APIs to a beginner software developer.

## Constraints:
- Limit response to 120–150 words
- Include one real-world analogy
- Provide one JSON example (request + response)
- Avoid technical jargon unless explained
- Use simple language (grade 6–8 level)
- End with a one-line summary

## Output Format:
- Paragraph explanation
- JSON example in code block
- Final summary line
